# Papers Folder

This folder categorizes research articles and literature referenced by the AETHEL clinical machine learning auditing framework.

## Subdirectory Structure

*   `literature_review/`: Broad surveys and general auditing reviews.
*   `ieee/`: Hardware, signal processing, and general engineering AI papers.
*   `medical/`: Journal articles from clinical venues (e.g. Lancet, JAMA, Nature Medicine).
*   `explainable_ai/`: SHAP, LIME, ALE, and other local/global feature attribution papers.
*   `trustworthy_ai/`: Algorithmic safety, reliability, and fairness surveys.
*   `calibration/`: Post-hoc recalibration methods (Platt, isotonic, beta scaling).
*   `robustness/`: Noise injection, feature ablation, and out-of-distribution stress.
*   `clinical_decision_support/`: Decision Curve Analysis (DCA), net benefit, and EHR integration.
*   `supplementary/`: Appendices, manuals, and data registry documents.
